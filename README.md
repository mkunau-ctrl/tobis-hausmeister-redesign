# Tobi's Hausmeisterservice — Redesign-Mockup

Multi-Page Redesign-Demo der Website [tobis-hausmeisterservice.com](https://www.tobis-hausmeisterservice.com).

Diese Version ist als **Köder-Mockup** gedacht — ein einmaliger Premium-Entwurf, um zu zeigen, was im Webdesign möglich ist. Die finale Website wird selbstverständlich an die Wünsche des Kunden angepasst (Texte, Farben, Bilder, Struktur).

## Seitenstruktur

| Seite | Datei | Zweck |
|---|---|---|
| Start | `index.html` | Hero + Drei Säulen + Zitat + CTA |
| Leistungen | `leistungen.html` | 5 Kernleistungen detailliert mit Bildern |
| Der Meister | `meister.html` | Persönliche Vorstellung + Werte |
| Referenzen | `referenzen.html` | Bildergalerie der Arbeit |
| Kontakt | `kontakt.html` | Telefon, E-Mail, WhatsApp + Formular |

## Designkonzept

**„Meisterwerk Editorial"** — der Elektromeister als Kunsthandwerker. Tiefes Waldgrün mit gebürsteten Messing-Akzenten, redaktionelles Magazin-Layout. Hebt sich bewusst von typischen Handwerker-Websites ab.

- **Fonts:** Fraunces (Display, variabel) · Manrope (Body) · JetBrains Mono (Akzente)
- **Palette:** Deep Forest `#0a1410` · Antique Brass `#c9a96e`
- **Stack:** Pures HTML/CSS/JS — kein Build, kein Framework, läuft überall
- **Bilder:** Unsplash CDN, austauschbar gegen eigene Fotos

## Anpassungen

Alle Inhalte sind in den HTML-Dateien direkt anpassbar. Bilder über die `src`-Attribute austauschen, Texte direkt im HTML editieren.

## Lizenz

Designentwurf. Bilder via Unsplash (frei verwendbar). Texte sind Platzhalter und können beliebig geändert werden.
